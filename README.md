# buscq.com Telegram bot
Real-time data from Santiago de Compostela buses, in a Telegram bot

### Required libraries

* [telebot](https://github.com/eternnoir/pyTelegramBotAPI)

### Usage notes

Replace the Telegram bot key in [buscq_bot.py](buscq_bot.py)
